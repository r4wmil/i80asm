# Лабораторные по ассемблеру i8085

## Нововведения:
- Простой запуск с отладкой
- Теперь программу можно скачать в любую папку на компьютере
- Правильное отображение файлов в современных редакторах (кодировка UTF-8)

## Л/р 1
1. Измените `lab\lab1.a85` в своём любимом редакторе
2. Запустите `lab1.bat`

## Л/р 2
1. Посмотрите `lab\L_01.A85` в своём любимом редакторе
2. Запустите `lab2.bat`
